## Propósito

El sitio web de **BOLD** tiene como objetivo presentar y promocionar los servicios de fotografía profesional en Puerto Rico. Busca transmitir la identidad visual de la marca, mostrar portafolios de trabajos realizados, facilitar el contacto con clientes potenciales y vender equipo de fotografía.

## Tecnologías

Las tecnologías utilizadas en este sitio web son las siguientes: 

- **HTML5**: Para la estructura del contenido.
- **CSS3** con **Bootstrap**: Para los estilos predefinidos. Se utilizó un framework para incluir componentes como un campo de e-mail, barra de navegación, tooltips, entre otros.
- **JavaScript (JS)**: Para la interacción dinámica en el sitio.
- **Figma**: Utilizado en la etapa de diseño de bocetos y prototipos de alta fidelidad antes de codificar

## Instalación y ejecución

Este proyecto no requiere instalación de dependencias ni el uso de Git.

### Instalación
1. Descarga o copia los archivos del proyecto en un folder local.
2. Verifica que la estructura del proyecto se mantenga intacta y sin errores (archivos `.html`, carpetas de `css`, `js`, multimedia, etc.).

### Ejecución
1. Abre el archivo principal `index.html`.
2. Ejecuta el archivo en cualquier navegador web moderno (Chrome, Firefox, Safari, Edge, Opera, etc.).
3. El sitio web se cargará automáticamente y estará listo para usarse. Ahora puedes cargarlo a un servicio de alojamiento como GoDaddy o Hostinger. 
> **Nota:** Se recomienda utilizar la versión más reciente del navegador para garantizar compatibilidad y correcto funcionamiento.
